== Description ==
This plugin is a easy to use addon for WPforms which helps to send form data to HubSpot lists with very less configuration and easy to use interface.

Documentation: <a href="https://wpoperation.com/wp-documentation/active-campaign-wpforms/">View Online</a>

If you are looking to integrate HubSpot with Contact Form 7 then here is plugin link: <a href="https://wordpress.org/plugins/wpop-contactform-hubspot/">Contact Form 7 to Hubspot by WPOP</a>

=== Features === 

Adds subscription in "HubSpot" through "WPForms"
Option to select "HubSpot" list ID for subscribers
Option to select "WPForms" fields for "HubSpot" list
Constant updates
Translation ready

== Installation ==

1. Unzip ac-wpforms.zip
2. Upload all the files to the /wp-content/plugins/wpop-wpforms-hubspot
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Install and activate WPForms plugin
